الفرق بين وسوم العناوين والفقرات في HTML

لماذا تتغير أماكن النصوص عند تغيير الوسوم؟

وسوم HTML المختلفة لها خصائص تصيير افتراضية في المتصفحات تحدد:

  • حجم الخط وسمكه
  • المسافات حول العنصر (الهامش والحشوة)
  • نوع العنصر (كتلة أو ضمني)
  • التسلسل الهرمي للعناوين

هذه فقرة عادية. الفقرات تستخدم للنصوص الطويلة والمحتوى الأساسي. لاحظ أن الفقرة تحتوي على مسافات أعلى وأسفل بشكل تلقائي، لكنها أقل من تلك الموجودة في العناوين.

هذا عنوان آخر من المستوى الثالث

ملاحظة هامة:

وسوم مثل <h1> - <h6> هي عناصر كتلة (block) تأخذ العرض الكامل وتسبب فاصل سطر قبل وبعدها.

الوسوم <p> هي أيضا عناصر كتلة ولكن بمسافات وخصائص مختلفة.

لا يوجد وسم <p1> أو <p2> في HTML، فوسوم الفقرات هي <p> فقط.

كيفية التحكم في المظهر

يمكنك استخدام CSS للتحكم في مظهر وسوم HTML لجعلها تظهر كما تريد:

h3 {
  margin: 10px 0;
  padding: 5px;
  background-color: #f0f0f0;
  border-right: 3px solid #3498db;
}

p {
  line-height: 1.6;
  margin: 15px 0;
}
تم إنشاء هذا المثال لتوضيح الفروقات بين وسوم HTML
